`install.sh` is a 173-line pure POSIX bash bootstrap. Every tool it invokes — `curl`, `wget`, `uname`, `sha256sum`, `shasum`, `sysctl`, `ldd`, `mkdir`, `chmod`, `rm` — is a standard system utility present on every supported target OS without installation. There are no third-party libraries, SDKs, or frameworks involved: the GitHub Releases URL pattern it constructs (`https://github.com/flora131/atomic/releases/…`) is a simple string convention, not an API backed by a client library. The JSON manifest is parsed with plain `bash` regex (`BASH_REMATCH`) and `sed`/`tr`, deliberately avoiding any `jq` dependency. Since the rewrite targets pi-coding-agent (not a different platform or language), the shell idioms stay identical; no external documentation is needed to remap any dependency.
