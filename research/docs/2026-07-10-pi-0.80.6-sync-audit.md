## Scope and method

The upstream comparison touched 97 coding-agent files (3,479 additions and 594 deletions across 47 path-touching commits). Atomic did not wholesale-copy those files. Each upstream source, test, documentation, example, README, and changelog change was compared with Atomic's split/refactored fork, then classified as an issue-required fix, dependency-inherited behavior, an existing Atomic equivalent, an intentional non-applicable divergence, or additive upstream feature work outside #1703's fix contract.

The implementation ports only applicable behavior and preserves Atomic branding, `@bastani/*` package names, the `atomic` CLI, `.atomic` paths, bundled first-party extensions, virtual-module compatibility aliases, versionless `0.0.0` manifests, Verbatim Compaction, hashline editing, the split standalone launcher/archive layout, lock-free auth reads, and Atomic-specific Copilot/Gemini model, context-window, retry, sanitization, and streaming repairs.
